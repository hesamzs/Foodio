<p id="description">Food reservation robot for the Samad system of the University of Guilan</p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Flutter</p>

```
flutter pub get
flutter run
```

<p>2. Python</p>

```
pip install python-bidi
pip install selenium
python3 Foodio.py
```
